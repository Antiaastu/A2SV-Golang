# Library Management System

## Features
- Add, Remove Books
- Add Members
- Borrow, Return Books
- List Available Books
- List Borrowed Books by Member

## Data Structure
- Books: Map[int]Book
- Members: Map[int]Member

## Error Handling
- Book not found
- Member not found
- Book already borrowed
- Book not borrowed by this member

## Console Commands
1. Add Book
2. Remove Book
3. Borrow Book
4. Return Book
5. List Available Books
6. List Borrowed Books by Member
7. Add Member
0. Exit
